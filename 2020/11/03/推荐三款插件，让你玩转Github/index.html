推荐三款插件，让你玩转Github

发表于 2020-11-03 分类于 杂谈

前言

作为Github的常客，真的是无法忍受那个速度，加载慢，下载慢，clone慢，总之慢慢慢……

除了慢，还有很多文件夹，层层包裹，你不觉得难受吗？

今天，作者推荐三个Chrome插件，让你喜欢上Github。

Octotree

Octotree这个插件能够为Github生成一个文件菜单树，类似IDEA目录结构，如下图：

那么这个插件如何安装呢？其实很简单，如果你的电脑可以翻墙，只需要在谷歌的应用商店下载即可。

但是，很多朋友没办法翻墙，只需要下载一个离线的插件包（octotree-chrome插件.crx），然后拖到谷歌的扩展程序中（chrome://extensions/）。

离线下载地址：Octotree下载

安装完成之后，在Github页面按住CRTL+SHITF+S快捷键则会出现文档菜单。

Github加速

Github加速这款插件能够显著提高git clone和下载打包文件的速度：

安装成功后，会在Github页面出现以下界面：

可以直接点击加速克隆和下载按钮，选择加速之后的URL进行下载。

安装方式同样有两种，一种是谷歌应用商店搜索安装（需要翻墙），另外一种则是下载离线安装包。

离线下载地址：Github加速下载

Github Downloader

一款很爽的插件，能够下载Github中的单个文件，没安装这个插件之前，只能先clone全部，安装这个插件之后，在每个文件上方都有一个Download按钮，如下图：

下载方式也是有两种，可惜我没找到离线包，所有只能翻墙安装了。

安装地址：Github Downloader

总结

三款插件的作者用了很久，公司同事都在使用，喜欢的可以装下尝尝鲜。
